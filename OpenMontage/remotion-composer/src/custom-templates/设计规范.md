# 模板视觉设计规范（全息科技风）

> 单一事实源：本规范描述的每条规则都已落到代码 SSOT，**改代码即改规范**。
> 禁止在各场景里内联颜色 / 辉光 / 标题动效，一律读 `useTheme()` 与
> `theme/surfaces.tsx`、`theme/textStyles.ts`、`primitives/HoloTitle.tsx`。

## 1. 风格定位

全息投影感（holographic）：**深色底 + 白字 + accent 霓虹辉光**。
画面元素（标题、元素框、Icon 片、强调光条）都带一层 accent 辉光，像投影出来的 HUD。

## 2. 颜色（`theme/palettes.ts`）

- 5 套主题统一为「深色科技底 + 白字」，仅 `accent` 与字体保留各自调性。
- 文字三档全白，仅靠不透明度区分层级（`TECH_TEXT`）：
  - primary `#FFFFFF`、secondary `#C8D2E0`、muted `#8A93A3`。
- `accent[]` 至少 4 色，按卡片顺序循环取用；`accent[0]` 为主辉光色。
- 语义色（如 callout 警示橙）是**刻意保留**的强调，不算违反白字规范。

## 3. 字体（`theme/palettes.ts` FONTS + `theme/tokens.ts` FONT_SIZE）

- 字号阶梯：caption 24 / body 28 / subtitle 36 / title 48 / display 72，硬下限 24。
- 标题统一 900 字重、负字距（display −1.5，title −1）。

## 4. 全息辉光令牌（`theme/tokens.ts` 的 `GLOW`）

「全息感」的总开关。改这里，全片辉光同步增减：

| token | 作用 | 默认 |
|---|---|---|
| `GLOW.text` | 标题文字辉光（弥散 + 暗描边保可读） | blur 18 / α 0.5 |
| `GLOW.border` | 元素框描边的常驻辉光基线 | blur 16 / α 0.22 |
| `GLOW.icon` | Icon / 序号方片的 accent 辉光 | blur 18 / α 0.3 |
| `GLOW.bar` | 强调光条（下划线 / 分隔）辉光 | blur 14 / α 0.6 |
| `GLOW.decodeFrames` | 标题解码入场时长（帧） | 20 |

通过 `useTheme().GLOW` 读取。

## 5. 元素框 / Icon / 光条（`theme/surfaces.tsx`）

- `techPanel` / `TechPanel`：玻璃底 + 霓虹描边 + 四角括号 + **常驻 GLOW.border 辉光**
  （`glow` 入参在此基线上叠加 frame 驱动的「呼吸辉光」）。
- `techIconChip`：科技小方块，自带 **GLOW.icon** accent 辉光。
- `accentUnderline(theme, {width, glow})`：accent 渐变光条，自带 **GLOW.bar** 辉光。
- `holoTextShadow(accent)`：标题文字辉光 text-shadow（供 HoloTitle 与发光标题用）。
- `glowBlob`：英雄场景身后的弥散光团。

## 6. 标题动效 SSOT（`primitives/HoloTitle.tsx`）

**所有标题同源**，片头主标题与右上角章节标题共用 `HoloTitle`，保证字体 / 动效 / 下划线一致：

- 入场：乱码解码（Scramble / Decode），时长 `GLOW.decodeFrames`。
- 文字：统一字体 + 900 + 白字 + `holoTextShadow` accent 辉光。
- 下划线：`accentUnderline` 渐变光条，随解码进度展开。
- 出场：投影成形（长阴影坍缩 + 淡出，可关）。
- 参数：`align`（center/right/left）、`size`（display/title）、`maxWidth`、`underlineWidth`。

用法：
- 右上角章节标题 → `SceneTitle`（仅定位，内部 `HoloTitle align="right" size="title"`）。
- 片头主标题 → `IntroScene`（`HoloTitle align="center" size="display"` + 背景全息 wash）。

## 7. 背景（`IntroScene` 等英雄场景）

深色底之上叠：`glowBlob` 弥散光晕 + 低透明度扫描线 wash（radial mask 收口），强化投影质感。

## 8. 新增 / 修改时的红线

1. 不在场景里写死颜色、辉光、标题动效；只读 SSOT。
2. 调「全息强度」改 `GLOW`；调字号/间距改 `tokens.ts`；调配色改 `palettes.ts`。
3. 新标题一律用 `HoloTitle`，不要再造一套入场动画。
4. 验证一律走 Remotion 预览（studio / Player），无明确指令不主动渲染（见 `AGENT_GUIDE.md`）。
